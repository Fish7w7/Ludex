# Design Direction

The first visual direction for Ludex is a usable desktop launcher screen, not a landing page.

## Direction

- Dark Japanese cyber arcade.
- Kawaii tech details used sparingly.
- Neon purple, cyan, and pink accents with a charcoal base.
- Rounded cards and panels.
- Smooth microinteraction-ready states.
- Sidebar-first desktop app shell.
- Dense library view with selected game details.
- Scanner and sync status visible but not noisy.

## First Concept

A simple generated concept was created before implementation to validate the direction. It includes:

- `Ludex` brand and `ピコ~` signature.
- Left sidebar.
- Library filters.
- Game cards.
- Selected game detail panel.
- Scan and sync cues.

The Phase 1 UI shell follows the broad structure but is not treated as the final approved UI.
